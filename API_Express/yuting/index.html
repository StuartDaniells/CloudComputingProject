Finding a Protein Motif

Using Uniprot and Regular expression in Python to find the motif pattern

Protein motif is a repetitive pattern inside a protein sequence string. Each protein family has the same motif, if a protein has the same motif sequence, then there is high chance that the protein has similar characteristics as other proteins inside that family. There are lots of open source to download certain protein sequences, e.g. uniprot. With the protein sequence, using regular expression we can easily identify if a protein belongs to a protein family.
